### Clone ###

Make sure you have a Git client on your machine
- [Windows] Install git-scm from:
  http://git-scm.com
- [Unix]
```bash
sudo apt install git
```

- Clone repo
```bash
git clone https://github.com/HiPeRT/hpc18.git
```
